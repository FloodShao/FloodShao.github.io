MyEclipse 安装

Posted on 2019-06-24 | In 技术 |

本安装仅供学习交流使用，如继续使用请支持正版购买。

安装系统： Mac

默认已经安装了Java jdk1.8，并设置好了环境变量。如果没有设置好，请参考mac jdk1.8安装
安装文件下载：
链接： https://pan.baidu.com/s/1dKuEMdzjwPitjxdzubTxZA
密码： 334l MyEclipse2017CI9-Mac

安装步骤

  1. 下载dmg文件进行安装,同时下载破解文件进行解压

  2. 等待安装完毕后，关闭MyEclipse进行破解

  3. 在/Applications/MyEclipse 2017 CI/MyEclipse 2017 CI.app/Contents/Profile/Plugins/中替换破解文件中的crack

  4. 双击运行cracker2017.jar 在username中随便输入，后边的professional改为blue。

  5. 双击SystemId生成注册信息，并激活Activate

  6. 左上角Tools点击保存activation信息, 1_save_properties

  7. 查看激活状态。状态栏Help->Subscription Information

Mac 上利用MyEclipse进行hadoop java开发

  1. 下载或自行编译hadoop-eclipse-plugin-2.9.2.jar。 一般搜索该jar包都能搜到，自行编译请参考https://github.com/winghc/hadoop2x-eclipse-plugin。自行编译需要线性安装ant，而且非常慢。我在网上下载的包可以使用，所以应该问题不大。

  2. 找到/Applications/My Eclipse/My Eclipse Drop-ins/dropins，将上述jar包放入dropins中。

  3. 重启MyEclipse，查看Preference, 是否存在Hadoop Map/Reduce选项。如果存在，则说明插件安装成功。


    如上图所示，添加hadoop安装目录。这里不一定要将hadoop安装目录解压到/usr/local/等目录，只要能够找到该目录就可以了。

  4. 如果只是在me上验证程序，不进行部署的话，到这里就可以了。如果需要在mac上进行hadoop部署的话需要进行后续。参考https://www.cnblogs.com/chaofn/p/4580554.html

  5. 添加一个map/reduce project。
    测试import org.apache.hadoop.io.IntWritable;
    如果能够找到，说明添加库成功。

MyEclipse设置自动补全

在Preferences下找到下图中红框的内容，原本只有一个’.’,表示只有填写’.’才会触发自动补全，我们要做到是添加所有字母’a-z’外加一个’.’，这样就可以在任何时候都触发自动补全。
